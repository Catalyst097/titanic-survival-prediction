  Titanic Survival Prediction 

This project predicts whether a passenger survived the Titanic shipwreck using Logistic Regression and Random Forest.

## Key Insights
- **Data Cleaning (90%):** Handled missing Age values and used One-Hot Encoding for categorical features.
- **Accuracy:** Achieved **82.09%** with Logistic Regression.
- **Lesson:** Simpler models sometimes generalize better than complex ones on smaller datasets!

##  Tech Stack
Python, Pandas, Scikit-Learn, Seaborn.
